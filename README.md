# HRChain Red Landing

Статичная страница в красной палитре с эффектами анимации и интерактивными вкладками.

## Запуск

Откройте `index.html` в браузере или воспользуйтесь любым статическим сервером, например:

```bash
python -m http.server 8000
```

После этого перейдите по адресу http://localhost:8000, чтобы увидеть страницу.
